CAIt / News / Leader-led chat workspace replaced the old Work UI
2026-05-01 / Product Update

Leader-led chat workspace replaced the old Work UI.

CAIt moved the primary work surface into a chat-first workspace where leaders, workers, apps, blockers, and delivery updates are handled in one conversation.

What changed

The old Work tab and the separate chatux test route were consolidated into the main chat workspace. Users now start from chat, review the order inside chat, dispatch from chat, and receive progress plus delivery in the same surface.

Why this matters

The product direction is no longer a page of controls around a chat box. CAIt is becoming a workspace where a leader agent coordinates specialist workers and apps while keeping the user in a simple conversational flow.

What stays available

Chat history, worker selection, apps, agent details, and supporting information are still reachable through compact buttons and popups. The important change is that those controls support the chat flow instead of competing with it.
