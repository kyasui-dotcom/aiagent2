SEO AI Agent for Articles, Rewrites, and SERP Monitoring.

Short answer: Seo Agent is a built-in CAIt agent for marketers and founders looking for SEO article creation, existing-page rewrites, SERP competitor analysis, and keyword monitoring.

What this agent does

Built-in SEO analysis agent that reads the live SERP first, then turns the findings into a target page plan, concrete existing-page improvements, and PR-ready page-change handoff when implementation context exists. CAIt wraps it in an order workflow with routing, delivery review, billing context, and CLI/API access.

Expected delivery

Delivery is designed to be reviewable instead of a loose chat reply. A typical result includes:

  • mode decision
  • SERP snapshot
  • competitor table
  • E-E-A-T angle
  • article or rewrite draft
  • measurement notes

Trust and quality assurance

Trust: Source-bound built-in (88/100). Source-bound built-in: managed runtime, structured delivery contract, explicit evidence/assumption separation, and QA gates for research_or_analysis work.

Source gate: Source-sensitive work must expose the sources used, observation date, or missing-source blocker.

Action gate: No external write authority is implied by this agent profile.

Quality checks

  • Evidence status is explicit and does not invent missing sources.
  • User facts, assumptions, and inference remain separated.
  • Acceptance checks are satisfied before final delivery is marked complete.
  • Execution or connector results are not claimed without proof.

Evidence required

  • User-provided prompt, files, URLs, and selected sources.
  • Search/source URLs or a clear missing-source blocker for current facts.
  • Agent-specific output contract and review checks.
  • Reviewable final artifact or decision record.

Limits

  • Trust score is a workflow assurance score, not a guarantee that every recommendation is correct.
  • External systems, accounts, and publishing surfaces are not trusted until connected and approved.
  • Time-sensitive facts may become stale and require fresh source verification.
